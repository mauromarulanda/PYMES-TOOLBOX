Kali Linux

Kali Linux is a Debian-based Linux distribution that focuses on various computer security tasks, such as penetration testing, security research, computer forensics and reverse engineering. Kali Linux contains a wide variety of tools and utilities to perform these tasks, from Aircrack-ng to Zaproxy.

Fluxion

Fluxion is a new tool for performing Wi-Fi wireless network audits. Fluxion works similarly to Wifite, it launches a social engineering attack and checks if the user of the attacked Wi-Fi network enters his network password and checks if the Wi-Fi password is correct or not.

Metasploit

Metasploit is an open source project for computer security, which provides information about security vulnerabilities and helps in pentesting and developing signatures for intrusion detection systems.

Nikto

is a free to use, open source (GPL) web server scanner that performs a vulnerability scan on web servers for multiple items, including dangerous files and programs, and looks for outdated versions of web server software. It also checks for server configuration errors and possible vulnerabilities that may have been introduced.

Lynis

Lynis is a battle-tested security tool for systems running Linux, macOS or Unix-based operating systems. It performs a comprehensive analysis of the state of your systems to support system hardening and compliance testing. The project is open source software licensed under the GPL and has been available since 2007.

Nmap

Nmap is an open source port scanning program originally written by Gordon Lyon, better known by his alias Fyodor Vaskovich, and now developed by a community. It was originally created for Linux but is now cross-platform. It is used to evaluate the security of computer systems, as well as to discover services or servers on a computer network, for which Nmap sends defined packets to other computers and analyzes their responses.
